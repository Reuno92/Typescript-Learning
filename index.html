Enum

New Type in Typescript : Enum

First Example :


enum Color {
Gray,
Green,
Blue
}

let myColor: Color = Color.Green;
console.log(myColor); // 1 :the index

Second Example :


enum Color {
Gray,
Green = 100,
Blue
}

console.log(Color.Green); // Return 100
console.log(Color.Blue); // Return 101
